=head1 NAME

cwb-itoa - Decode uncompressed binary CWB data file to ASCII numbers

=head1 SYNOPSIS

B<cwb-itoa> [-nl] I<file>

=head1 DESCRIPTION

B<TODO>

Reads 32-bit integers in network format from CWB binary data file I<file> or from standard input and prints the values as ASCII numbers on standard output (one number per line).


=head1 OPTIONS

Please note that no CWB programs currently support GNU-style C<--long-options>.

  -n  read integers in network format [default]
  -l  read integers in little endian format
